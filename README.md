# Data-Cleaning-and-Visualization-Project

## Project Overview
This project focuses on cleaning, processing, and visualizing a retail sales dataset using Python. The objective was to transform raw data into meaningful insights through data cleaning techniques and exploratory data analysis.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Workflow
- Data Loading
- Data Exploration
- Handling Missing Values
- Removing Duplicates
- Data Type Conversion
- Outlier Detection and Handling
- Data Visualization
- Dashboard Creation

## Visualizations
- Sales by Category
- Sales by Region
- Top 10 States by Sales
- Top 10 Customers by Sales
- Monthly Sales Trend
- Sales Distribution
- Sales Dashboard

## Key Findings
- Identified the highest-performing product category.
- Analyzed regional sales performance.
- Identified top-performing states and customers.
- Observed monthly sales trends.
- Visualized sales distribution for better business understanding.

## Conclusion
This project demonstrates how data cleaning and visualization techniques can convert raw data into actionable business insights.

## Dataset
Sample Superstore Dataset (downloaded from Kaggle)
